Contract Keyword Search

Search a whole folder of contracts for the clauses or terms you care about, and get back a color-coded Excel grid showing which contracts contain what — with short AI summaries of each match.

Download the script Read the instructions →

What it does

Search hundreds of contracts at once for specific clauses — outputs a green/red Excel grid plus AI summaries.

What you need

These tools run on Python, a free program. You install it once, then it powers every Gavel tool. If you've already set up Python for another Gavel tool, skip straight to the run step.

  1. Install Python (one time)

    Download the free installer and run it. On Windows, tick "Add Python to PATH" on the first screen.

    Mac
    Download from python.org/downloads
    Windows
    Download from python.org/downloads

  2. Open your command line

    This is the plain text window where you type commands. It's already on your computer.

    Mac

    Open Terminal (Applications → Utilities).

    Windows

    Open Command Prompt (search "cmd" in the Start menu).

  3. Install the libraries this tool needs

    Copy the line for your system, paste it into the command line, and press enter. This downloads the small add-ons the tool uses.

    Mac
    Terminal
    pip3 install anthropic openpyxl pypdf python-docx
    Windows
    Command Prompt
    pip install anthropic openpyxl pypdf python-docx

Set your API key · one time

This tool uses Claude AI to read and describe documents, so it needs a free API key from Anthropic. This is separate from a ChatGPT/Claude.ai subscription — it's a developer key. You set it once.

  1. Get a key

    Sign up at console.anthropic.com, go to API Keys, and create one. Copy it (it starts with sk-ant-).

  2. Tell your computer the key

    Paste the matching line into your command line, swapping in your real key. This stores it for the current session.

    Mac
    Terminal
    export ANTHROPIC_API_KEY="sk-ant-your-key-here"
    Windows
    Command Prompt
    set ANTHROPIC_API_KEY=sk-ant-your-key-here
Note on cost: Anthropic charges a small per-use fee for API calls (usually a few cents per document). New accounts include free starter credit. Your documents are sent to Anthropic's API only to generate the descriptions.

Step by step

This assumes zero technical experience. Follow the steps in order — you only do the Python setup above once.

  1. Put the contracts in one folder

    Create a folder called contracts with the PDFs or Word files you want searched, and save contract_search.py in the same place.

  2. Add your AI key

    This tool uses Claude to summarize matches. Follow the "Set your API key" box above once.

  3. Enter your keywords and run

    When it runs, type the clauses or terms you're looking for (e.g. indemnification, arbitration, non-compete). It builds contract_search.xlsx — a green/red grid showing which contracts contain which terms, plus summaries.

The command to run it

Open your command line, navigate into your document folder, then paste this:

In your document folder
python contract_search.py
Tip — how to "navigate into your folder": type cd (with a space), then drag the folder from your desktop into the command-line window and press enter. Now the run command above will find your files.

Common errors & fixes

If you see…Do this
AuthenticationError / invalid API keyRe-do the "Set your API key" step and confirm you pasted the entire key.
ModuleNotFoundError: anthropicRun the install command above, then try again.
A scanned contract shows no textScanned image-only PDFs have no readable text. Run them through an OCR step first, or search those by hand.
